# Zipo — Dr Marcus GP Clinic Website Assistant

**Zipo** is a navigation and routing chatbot embedded on Dr Marcus Coelho's clinic website.

**Clinic:** Kiama Downs Medical Practice
**Address:** 31 Johnson Street, Kiama Downs NSW 2533
**Phone:** (02) 4237 7502
**Email:** info@kdmp.com.au

## What Zipo Does

- Guides patients to the correct booking system
- Directs patients to official clinic links
- Provides trusted medical information from the approved knowledge base
- Reduces booking confusion and improves website usability

## What Zipo Cannot Do

- Does NOT provide medical advice or diagnoses
- Does NOT collect or store personal data
- Does NOT interpret test results or handle urgent medical concerns

> **EMERGENCY NOTICE:** This service is not suitable for urgent or emergency medical issues. If this is an emergency, call **000** or attend your nearest Emergency Department.
